# Acika Label - Frontend (React + Vite + Tailwind)

Scaffolded project to reproduce the provided homepage design.

Quick start:

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

Notes:

- Replace placeholder images in `src/pages/Home.jsx` and `src/components/Hero.jsx` with your high-resolution assets for pixel-perfect results.
- This scaffold includes routes for main menu items; fill the placeholder pages in `src/App.jsx`.
